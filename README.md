# Manufacturing Flow Designer — PWA V4.0.002

This is a testable engineering build focused on the process-canvas foundation.

Implemented and testable:
- Fixed full-screen application shell
- Resource library with hand, automatic/CNC, walking and inspection icons
- Add resources by tap or drag
- Move resources on the process canvas
- Mouse-wheel zoom and touch pinch zoom
- Pan on empty canvas space
- Eight connection ports per resource
- Orthogonal routed links with high-contrast arrowheads
- Select links and delete them with the dedicated Delete Link tool
- Save locally, export JSON project files and reopen them
- Standard Work, Factory Layout, Simulation and Analytics workspaces are included as early functional previews
- Offline PWA manifest and service worker

Run through HTTPS or localhost. Opening index.html directly may restrict service-worker and file features.
