# V4.0.001 PWA Test Guide

## Recommended setup
Serve this folder through GitHub Pages or a local web server, then open it in Safari, Chrome or Edge.

## Process canvas tests
1. Tap a resource in the library and confirm it appears on the canvas.
2. Drag a library resource onto a chosen canvas position.
3. Drag each process card and confirm attached links follow it.
4. Pan by dragging empty canvas space.
5. Zoom with the mouse wheel, or pinch with two fingers on iPad.
6. Confirm the browser page itself does not scroll while manipulating the canvas.

## Link tests
1. Select **Link**.
2. Choose a source connection point and then a destination point.
3. Confirm the connector uses a 90-degree route and its arrowhead is visible.
4. Select **Delete Link**.
5. Tap or click a highlighted link and confirm it is removed.
6. Confirm the tool returns to Select mode after deletion.
7. Right-click a link on desktop, or long-press it on touch, to test direct deletion.

## Save/open tests
1. Move resources, create links and choose non-centre ports.
2. Use **Export** to save the project.
3. Refresh or reopen the application and import the project.
4. Confirm card positions, links and selected ports are preserved.

## Preview workspace tests
Open Standard Work, Factory Layout, Simulation and Analytics. These are included for early feedback, but Build 001 acceptance is based primarily on the process canvas and link tools.

## Report an issue
Record the device/browser, exact steps, expected result, actual result and a screenshot where possible.
